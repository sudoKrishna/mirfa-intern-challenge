# Evaluation Rubric (Internal)

## Auto Reject
- no deployment
- no Loom
- no tests
- encryption incorrect

## Good
- clean repo structure
- correct AES-GCM usage
- validations present
- tests written
- readable code

## Excellent
- strong architecture
- good error handling
- thoughtful explanation
- clear commits
- good reasoning about security

## Bonus
- DB persistence
- nice UI polish
- strong documentation
